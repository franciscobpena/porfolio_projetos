Desenvolvimento de um Painel Gerencial para Negócio com Streamlit

Análise Exploratória de Dados (EDA) do Mercado de Aluguel de Imóveis no Brasil

Neste projeto, realizamos uma Análise Exploratória de Dados (EDA) para entender o mercado de aluguel de imóveis no Brasil. O foco foi identificar padrões de preços, tamanhos de propriedades, número de vagas de garagem, e fatores como localização e proximidade a estabelecimentos comerciais. A análise gerou insights sobre a variação dos preços por bairro, tipo de imóvel e outras características importantes, ajudando a orientar decisões no setor imobiliário.

Análise Exploratória de Dados (EDA) de uma Empresa de E-commerce (Olist) utilizando SQL

Realizei uma análise detalhada em SQL segmentando clientes por avaliações, frequência de compra e categorias de produtos. Os principais insights indicam que, apesar da alta satisfação, a retenção de clientes está em queda, exigindo foco em estratégias de fidelização. Produtos de alto valor, como tecnologia, oferecem oportunidades para cross-selling, e otimizar os prazos de entrega é crucial para manter a satisfação do cliente.
